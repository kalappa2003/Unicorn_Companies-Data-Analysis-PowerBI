## Unicorn-Companies-Data-Analysis-PowerBI

## Project Objective
Analyzed global unicorn startups dataset using Power BI to identify trends in valuation, funding, industries, and geography. Built an interactive dashboard with KPIs, data modeling, and DAX to generate business insights.

## Questions(KPIS)
● Total number of unicorn companies
● Year-wise growth of unicorn startups
● Top countries producing unicorns
● Industry-wise distribution of unicorns
● Funding vs valuation analysis
● Top investors in unicorn companies

## Process
● Cleaned and transformed data using Power Query
● Handled missing values and standardized formats
● Built data model and relationships
● Created calculated measures using DAX
● Designed interactive dashboard with filters and slicers

## Project Insights
● Unicorn growth increased rapidly after 2015
● Technology and FinTech dominate the unicorn space
● The US and China lead in unicorn creation
● Higher funding does not always guarantee higher valuation
● Few investors appear repeatedly across multiple unicorns

## Final Conclusion
●Unicorn companies are driven by innovation, funding strategy, and market demand. Countries with strong startup ecosystems and investor networks produce the most unicorns.
